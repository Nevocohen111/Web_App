* A full-fledged stand-alone Spring boot application,hosted on  a heroku external server.
* Spring projects used on this application are:
1.Spring data JPA.
2.Spring security.
3.Spring boot thymeleaf.
4.Spring boot mail.
5.Spring lombok.
6.Spring Validation.
7.Spring boot Devtools.
8.Spring Web.
* Login in order to see the full functionality of the application.
1.Register to the website in order to access to full content of the website.
2.Registrating to the website also allows you to use the rest Api's services using Postman or any other technology to fetch data.
3.You can always register a new account(Will be created with the permissions of a user).
* Enjoy!
